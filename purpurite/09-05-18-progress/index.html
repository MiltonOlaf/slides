Project

Un proyecto en el contexto de Purpurite es un sitio web único. Por lo general, es una carpeta que contiene toda la información necesaria para construirlo. Dentro de esa carpeta hay posts, pages, datafiles, así como el archivo de configuración.

Estructura de directorios

Es importante comprender cómo se estructuran los datos en el sistema de archivos. Un proyecto de Purpurite es una carpeta en el sistema de archivos con un archivo de configuración y una estructura bien definida.
