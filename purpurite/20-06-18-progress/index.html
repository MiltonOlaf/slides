Metas

  • Crear un proyecto de calidad, competitivo, con un lugar entre las preferencias de la comunidad de generadores estáticos.
  • Convertir las edición de contenido estático en algo fácil y flexible.
  • Mantenerlo interesante para todo tipo de usuario, desarrolladores, diseñadores web y creadores de contenido con conocimientos menos avanzados.

Servidor HTTP

Basado en la petición GET

Un servidor web opera mediante el protocolo HTTP, de la capa de aplicación del Modelo OSI. Las peticiones al servidor suelen realizarse mediante HTTP utilizando el método de petición GET, en el que el recurso se solicita a través de la url al servidor Web.
