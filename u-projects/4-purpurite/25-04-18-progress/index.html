¿Qué es purpurite?

Cuando se trata de crear sitios web, hay una cantidad ridícula de herramientas disponibles. Van desde soluciones de gestión de contenido completas como Wordpress, soluciones de escritorio como Google Web Designer, soluciones de sitios web alojados en la nube como WIX o enfoques más centrados en programadores como Jekyll que generan sitios web de plantillas y archivos markdown.

Purpurite es diferente a cualquiera de estas.

Purpurite es un generador estático

La gran mayoría de los sitios web se leerán muchas más veces de las que se actualizarán. Esto es crucial porque el contenido dinámico no es gratis.

Purpurite es un CMS

Purpurite proporciona una interfaz de administrador flexible desde un navegador, desde la que puede editar los contenidos de su sitio web. A diferencia de las soluciones CMS tradicionales, purpurite se ejecuta completamente en su propia computadora.

Purpurite es un framework

Purpurite es mucho más que un generador de sitios web porque se basa en una base de datos de archivos planos internos muy flexible que se puede usar para modelar cualquier contenido de un sitio web.

Purpurite es "multi-usuario"

La colaboración puede basarse en sistemas de control de versiones como git o simplemente en soluciones básicas de alamacenamiento en la nuebe. Está construido intencionalmente para que la colaboración pueda funcionar a través de los sistemas muy básicos como Dropbox o con solo recursos compartidos de red.

CLI

Es un método que permite a los usuarios dar instrucciones a algún programa informático por medio de una línea de texto simple. 


~ $ purpurite serve
Generating...
Server address: http://localhost:8000/
Server running... press ctrl-c to stop.

~ $ purpurite --help
Options:
  serve            generate site on local server
  --help           Show this message and exit.
