Marco de desarrollo para soluciones IoT + ML basadas en microservicios

Milton Olaf Paredes

El IoT está impregnando lentamente cada aspecto de nuestras vidas...
Pero actualmente, se requiere una gran cantidad de entradas manuales para lograr una funcionalidad óptima. No hay mucha inteligencia incorporada.

El mercado de los sistemas IoT inteligentes esta dominado por las principales compañias de tecnologica, hasta ahora no hay suficientes herramientas abiertas disponibles para los pequeños desarrolladores de está tecnologia.

Desarrollo de un marco de trabajo open source, sobre el cual construir soluciones IoT impulsadas por modelos de aprendizaje profundo, sobre una arquitecturas de microservicios.
